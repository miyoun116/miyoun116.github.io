A Child's Delight

Responsive web design / 2017 Fall

Overview

This was a course project that involved redesigning the website for A Childâ€™s Delight into a responsive web design.

My Role

Designed a logo and main features to streamline shopping process. Created responsive layouts that looked simpler and more modern, and added category images on the drop-down menu to make it easier for users to find products.

Design Problem of The Original Website

Consistency

Navigation buttons and call to action buttons lack consistency.

Efficiency

It is difficult to find an item due to lack of filtering.

User comprehension

The background color is too strong, and it distracts users from the main content. Buttons on the navigation bar and product details are difficult to read because of small fonts and narrow spacing. It is not easy to find categories because too many categories are shown at once.

User Control and Freedom

There are no breadcrumbs which make the website easier to navigate.

Design Goal

To make a website that is easy to find items of interest, provide useful information about items and services, and is efficient to purchase.

Design Objective

Consistency: Consistent design on buttons to help users to recognize them.

Efficiency: Filtering by skills, pricing, etc. to make products search easier.

User comprehension: Consistent color systems, concise menu hierarchy with types of toys, interests and age, intuitive buttons on the navigation bar, and itemization in product details to make it easier for users to recognize and understand the contents.

User Control and Freedom: Breadcrumbs on the top of pages to let users know the current location on the site.
