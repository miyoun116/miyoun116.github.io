Penguin Jump
Overview

Penguin Jump is the theme for an iPad game using the same game play components as Angry Birds, with penguins and seals as the good and bad guys, respectively.


Timeline

Feb. 2019


Project info

A class project


My role

Graphic & UI design


Tools

Illustrator

Goals

Design characters and user interface of a mobile game, including layout, buttons, icons, and other visual elements that will enhance the player's experience.

Visual design
Characters

The main character is a penguin. The launch device is a ski jumping hill that the slope value is controlled by users and a penguin skies down and jumps using this.

The bad guys are seals who are on a pirate ship. A jumping penguin unbuilds a pirate ship and catches seals on the sea.
