Penguin Jump

Game Screen Design / 2019 Spring

Overview

As part of a school project, I created a mobile game screen called Penguin Jump, which utilizes similar gameplay components to Angry Birds while incorporating its own unique theme. The main characters are penguins and bed guys are seals. The launch device is a ski jumping hill that the slope value is controlled by users and a penguin skies down and jumps using this. A jumping penguin unbuilds a pirate ship and catches seals on the sea.

My Role

Designed the characters and user interface of a mobile game, including layout, buttons, icons, and other visual elements that will enhance the player's experience.
